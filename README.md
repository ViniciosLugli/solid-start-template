# solid-start-template

## Prerequisites

-   Node.js >= 20

## Installation

```bash
npm install
```

## Dev

To start the development server

```bash
npm run dev
```

## Building

Build the app

```bash
npm run build
```

Run the builded app

```bash
npm run start
```
